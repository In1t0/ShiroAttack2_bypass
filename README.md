## 工具特点

* javafx
* 处理没有第三方依赖的情况
* 支持多版本CommonsBeanutils的gadget
* 支持内存马
* 采用直接回显执行命令
* 添加了更多的CommonsBeanutils版本gadget
* 支持修改rememberMe关键词
* 支持直接爆破利用gadget和key
* 支持代理
* 添加修改shirokey功能（使用内存马的方式）**可能导致业务异常**
* 支持内存马小马
* 添加DFS算法回显（AllECHO） 
* 支持自定义请求头，格式：abc:123&&&test:123
* 通过在payload中随机添加字符来bypass WAF
* 支持Z-Godzilla_ekp内存马。
